# cloudnewpracticals
A comprehensive collection of Cloud Computing practicals covering AWS services such as EC2, Auto Scaling, Load Balancer, and cloud fundamentals. This repository demonstrates hands-on implementation of core cloud concepts aligned with academic curriculum.
🌩️ Cloud Computing Practicals

This repository contains all the practical experiments performed as part of the Cloud Computing course. It includes step-by-step implementations, configurations, and outputs using leading cloud platforms like AWS.

🎯 Objectives
Understand cloud computing concepts and architecture
Gain hands-on experience with cloud platforms
Learn to deploy and manage virtual machines
Configure scaling and load balancing
Explore cloud storage, networking, and pricing models

🛠️ Technologies & Services Used
Amazon Web Services (AWS)
EC2 (Elastic Compute Cloud)
S3 (Simple Storage Service)
Auto Scaling
Elastic Load Balancer (ELB)
VPC (Virtual Private Cloud)
Basic exposure to:
Microsoft Azure
Google Cloud Platform (GCP)

📚 List of Practicals
1️⃣ Introduction to Cloud Platforms
Created free-tier accounts (AWS, Azure, GCP)
Explored dashboards and services
Understood pricing models

2️⃣ Launch Amazon EC2 Instance
Created and configured EC2 instance
Connected using SSH
Deployed basic applications

3️⃣ Working with AWS Storage (S3)
Created S3 bucket
Uploaded and managed files
Configured permissions and access

4️⃣ Auto Scaling Configuration
Created Launch Template
Configured Auto Scaling Group
Set scaling policies (CPU-based scaling)

5️⃣ Load Balancer Setup
Created Application Load Balancer (ALB)
Configured target groups
Distributed traffic across instances

6️⃣ VPC and Networking
Created custom VPC
Configured subnets (public/private)
Setup internet gateway and routing
